# Support

## Questions and bug reports

- Use GitHub Issues for reproducible bugs and feature requests.
- Use the public docs first for setup, UI values, and lane boundaries:
  - `README.md`
  - `docs/HUMAN_QUICKSTART.md`
  - `docs/AGENT_QUICKSTART.md`
  - `docs/UI_VALUE_NOTES.md`
  - `docs/PUBLIC_BOUNDARIES.md`

## Security issues

For security-sensitive reports, do not post exploit details in a public issue.

Follow `SECURITY.md` instead.
